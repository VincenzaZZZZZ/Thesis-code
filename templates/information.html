Information

This is a web application for learning how to program in Python by debugging.
There are three levels: easy (caterpillar) - medium (cocoon) - hard (moth).
The level icon is initially black and white. After you pass it, it will become colourful.
If you pass the hard level, you will get a moth as a badge ,and you can see them on the badges page.
The points are calculated depending on level you passed. You will get 10 points for easy levels, 20 points for medium levels and 30 points for hard levels. If you use the 'HINTS' button, 3 points will be deducted for that level. Every level will have one hint and the points will be only deducted once each level.
The points are used to get the rank on the leaderboard. The leaderboard will show the top 5 students. During the debugging, you are expected to find the location of the bug, give the input(s) that leads to incorrect output and correct the given code to make it run properly. If you think your answer is correct, you can use the 'Report a problem' button to upload your answer to let the teacher manually check.
